About this project

lesser-dannatt is a personal project by GreekFellows.

The data are from a digitized version of Geir Tómasson Zoëga's A Concise Dictionary of Old Icelandic, kindly provided by Tim Ermolaev (stridmann at gmail dot com).

Any bug reports, comment, feedbacks, suggestions are absolutely welcome. Please feel free to start an issue on the GitHub page for this project.

Help

Search Options

These options can be enabled/disabled by checking/unchecking the checkboxes below the search box on the Search page. Options with the icon are experimental. Options with the icon are under development.
Perfect matches only

Only displays entries with one or more headwords matching exactly with the search input.

Example: Searching for af with this option disabled gives you not only the preposition af, which is probably what you are looking for, but also many other irrelevant entries since many words begin with af-. With this option enabled, you will only get the preposition af.

ASCII-friendly inputs 

Accepts long vowels without the acute diacritic, d as ð, t as þ, o for either of ö and ø. Additionally, replaces occurrences of ae with æ, oe with œ in the search input.

Examples: searching for foerda with this option enabled gives matching results for fœrda and fœrða as well as the original search entry.

Umlaut (mutation) alternations 

In addition to searching for the original search input, searches for the U-umlauted (back mutation) and I-umlauted (front mutation) forms of the input, if the input could be umlauted, and for the forms of the input prior to U-umlaut and I-umlaut, if the input appears to have undergone umlaut.

Examples: searching for handar with this option enabled also gives search results for höndur (U-umlauted) and hender (I-umlauted).
